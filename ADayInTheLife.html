A Day In The Life


I designed and programmed A Day in the Life in 48 hours for the Global Game Jam 2014 weekend event using C# and Unity. My friend Alex Hayes drew the art.


The theme for the event was "We don't see things as they are, we see them as we are." and so we created a game were an individuals morning bus run was different based on their attitude. The objective is to reach the bus before it leaves, avoiding puddles and people on the way. I programmed this game in C# using the Unity game engine.
