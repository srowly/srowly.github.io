Updater/Launcher


I created an updater/launcher as a personal project to send to friends, allowing them to download the latest version of any game or software I am producing. The launcher/updater can be easily modified to work with any program, the pictures above show the launcher/updater modified to work with my game "Space".


This project used C# and Windows Forms. It can be downloaded below, and when ran will download an older standalone version of my web game "Space" to exemplify its use.
